> Real-time Object Detection: Implementation of Single Shot MultiBox Detector in PyTorch.

## Install

```sh
conda env create -f environment.yml
```

## Usage

```sh
python train.py
```

## Run tests

```sh
python test.py --resume <path/to/model.pth>
```
